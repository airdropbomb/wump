## Installation

1. Clone the repository:

```bash
git clone https://github.com/airdropbomb/wump.git && cd wump
```

2. Install dependencies:

```bash
npm install
```

3.Add bearer Token
```bash
nano tokens.txt
```

4. Fil your wallet address on script

```bash
nano wump.js
```
